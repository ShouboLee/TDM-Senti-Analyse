项目包含4个子目录
- Dict
- ML 
- DL
- BERT
  
分别表示基于情感词典、SVM、LSTM和BERT的文本情感分类。

运行每个子项目的main.py文件启动各项目。

环境
- Python 3.7.10
- PyTorch 1.9.0 1.4.0
- Jieba 0.42.1
- Scipy 1.3.1
- Sklearn 0.21.3
- Genism 4.1.2

数据集：[weibo_senti_100k](https://github.com/SophonPlus/ChineseNlpCorpus/blob/master/datasets/weibo_senti_100k)
